# Blogging platform front

## Initialisation

```bash
git clone 
```

### Go to project folder

```bash
cd blogging-platform-front
```

### Create local env file

```bash
cp .env.example .env
```

### change .env

#### Enter your back api url

```dotenv
VITE_APP_BACK_URL=http://127.0.0.1:8000/api
```

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run dev
```
